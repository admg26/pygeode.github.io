Reference¶

Fundamental PyGeode objects¶

Var class overview

The fundamental class Var for representing a field on a geophysical grid, and many of the basic operations.

Axis class overview

The fundamental class Axis and its children which represent one dimension of the geophysical grid, and a description of all built-in types of Axis objects.

Dataset overview

A collection class Dataset to manage a set of Var objects.

Plotting package¶

Plot module

A module for plotting Var objects.

Top-level analysis¶

File Input and Output

File input and output routines.

Timeutils module

A module of routines for working with time axes

Climat module

A module of routines for computing climatological means and variances.

EOF Analysis

A module of routines for computing empirical orthogonal functions, singular value decompositions, and related variance decompositions.

Stats module

A module of routines for computing correlations, regressions, and computing statistical tests.

View module

A helper class for mapping subsets of variables
